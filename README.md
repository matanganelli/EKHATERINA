# EKHATERINA Website

A simple semantic HTML website for the indie pop project **EKHATERINA**.

This project was created to practice fundamental web development concepts using **basic HTML structure and semantic elements**.

## Features

* Semantic HTML structure
* Navigation menu
* Images and cover section
* Embedded Spotify player
* Embedded YouTube video
* External links to music platforms

## Technologies Used

* HTML5

## Project Structure

```
index.html
```

## Purpose

The goal of this project is to demonstrate the use of:

* semantic HTML elements
* page structure (`header`, `nav`, `main`)
* media embedding
* external links
